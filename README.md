# SPK Scoring

## 📌 Description
SPK Scoring is a web-based Decision Support System (DSS) developed to assist in evaluating and selecting the best alternatives using the Scoring Method. The system calculates scores based on predefined criteria and weights to generate objective and accurate decision results.

This application is built using CodeIgniter 3 and MySQL with a simple, responsive, and user-friendly interface.

---

## 🚀 Features
- Alternative data management
- Criteria and weight management
- Automatic scoring calculation
- Ranking and decision results
- Responsive dashboard interface

---

## 🛠️ Technologies Used
- PHP
- CodeIgniter 3
- MySQL
- Bootstrap
- JavaScript

---

## 🎯 Purpose
This system aims to simplify the decision-making process by providing accurate, fast, and transparent scoring results.
